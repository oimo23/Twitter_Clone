勉強用に作ったTwitterクローンアプリ。  
    
- 新規登録・ログイン・フォローアンフォロー・タイムライン・画像のアップロード・アップロード時の切り取り・ダイレクトメッセージなど
- バックエンドにはFirebaseを使用
- MVVM
- CocoaPods
- iOS13.0以上
- No StroryBoard
